## Get started in minutes

<div class="landing-quickstart">
  <div class="landing-quickstart__copy">
    <p class="landing-quickstart__lead">
      Install the package, drop one attribute, and let the registry hydrate every supported pattern. You can progressively enhance
      existing markup or scaffold new components from our playgrounds.
    </p>
    <ol class="landing-quickstart__steps">
      <li>Install the package and include the <code>auto-init</code> helper.</li>
      <li>Annotate triggers with <code>data-automagica11y-toggle</code> (or a context shortcut).</li>
      <li>Drop your markup into the docs playgrounds to validate keyboard, ARIA, and reduced-motion states.</li>
    </ol>
  </div>
  <div class="landing-quickstart__install">
    <h3>Install in seconds</h3>

```bash
npm install automagica11y
```

    <p class="landing-quickstart__note">
      Once installed, add `data-automagica11y-*` attributes to your markup and let the registry initialize everything automatically.
    </p>
  </div>
</div>
