# Meet automagicA11y

automagicA11y is a lightweight, framework-agnostic toolkit that turns semantic HTML and data attributes into fully wired accessible patterns. Drop an attribute, ship the ARIA.

## Install in seconds

```bash
npm install automagica11y
```

Once installed, add `data-automagica11y-*` attributes to your markup and let the registry initialize everything automatically.
